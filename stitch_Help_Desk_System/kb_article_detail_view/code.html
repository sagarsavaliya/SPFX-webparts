How to Configure VPN for Remote Access

Sarah Jenkins
Updated 2 hours ago Version 1.2

This guide provides step-by-step instructions for configuring the corporate Virtual Private Network (VPN) client on Windows and macOS devices. Following these steps ensures secure access to internal resources while working remotely.

info

Prerequisite

Before proceeding, ensure you have your Multi-Factor Authentication (MFA) token ready, as it will be required during the initial setup process.

1. Installing the VPN Client

To begin, you must download the authorized VPN client from the software center. Do not use third-party VPN clients as they are not supported by our security protocols.

  1. Navigate to the Software Center portal.
  2. Search for "SecureConnect VPN Client".
  3. Click Install and wait for the process to complete.
  4. Once installed, launch the application from your Start Menu or Applications folder.
Figure 1: Software Center Download Page

2. Configuring Server Settings

After launching the client, you will need to input the server gateway address manually if it does not auto-populate.

Use the following configuration parameters:

Gateway Address: vpn.enterprise-access.com
Port: 443 (Default)
Protocol: SSL-VPN

3. Troubleshooting Common Errors

If you encounter "Error 403: Access Denied", please verify the following:

  • Your network password has not expired.
  • You are not connected to another VPN simultaneously.
  • Your internet connection is stable.

For persistent issues, please check the related articles section or submit a ticket with the error log attached.
